# Personal Website

This is a personal website built with Next.js, TypeScript, Tailwind CSS, and Sanity.

## Technologies

- [Next.js](https://nextjs.org/) - React framework for production
- [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework
- [Sanity](https://www.sanity.io/) - Headless CMS for structured content

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v20 or later)
- npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/Personal-Website.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Running the Website

To run the development server for the website:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Sanity Studio

The Sanity Studio is located in the `sanity/personal-website` directory.

### Running the Sanity Studio

1. Navigate to the Sanity project directory
    ```sh
    cd sanity/personal-website
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run the Sanity Studio
    ```sh
    npm run dev
    ```

Open [http://localhost:3333](http://localhost:3333) with your browser to see the Sanity Studio.
